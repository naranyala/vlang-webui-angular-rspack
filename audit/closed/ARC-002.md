# Architecture Finding: ARC-002

## No Graceful Shutdown

**Severity:** MEDIUM  
**Impact:** Data loss, orphaned resources on shutdown  
**Status:** [DONE] **RESOLVED**

---

## Resolution

### Fixed In
- `src/window_manager.v` - ShutdownHandler and AppLifecycle
- `src/main.v` - Lifecycle integration

### Changes Made

1. **Created ShutdownHandler:**
```v
pub struct ShutdownHandler {
pub mut:
    handlers        []fn ()
    running         bool
    shutdown_requested bool
}

pub fn (mut sh ShutdownHandler) register(handler fn ()) {
    sh.handlers << handler
}

pub fn (mut sh ShutdownHandler) execute_shutdown() {
    println('Executing shutdown handlers...')
    
    for handler in sh.handlers {
        handler()
    }
    
    println('Shutdown complete')
}
```

2. **Created AppLifecycle manager:**
```v
pub struct AppLifecycle {
pub mut:
    window_manager  &IWindowManager
    shutdown        &ShutdownHandler
    initialized     bool
    start_time      u64
}

pub fn (mut al AppLifecycle) run(initial_page string) ! {
    if !al.initialized {
        return error('Application not initialized')
    }

    println('Application starting...')
    al.window_manager.show(initial_page)!
    
    // Block until window is closed
    al.window_manager.wait()
    
    // Execute shutdown handlers
    al.shutdown.execute_shutdown()
    
    println('Application exited')
}
```

3. **Updated main.v with graceful shutdown:**
```v
// ARC-002: Setup graceful shutdown handler
mut lifecycle := window_manager.new_app_lifecycle(mut window_mgr)
lifecycle.init() or {
    vlog('ERROR: Failed to initialize lifecycle')
    return
}

// Register cleanup handlers
lifecycle.on_shutdown(fn () {
    vlog('Cleaning up event bus...')
    event_bus.print_debug()
})

// ARC-002: Run with graceful shutdown
lifecycle.run('index.html') or {
    vlog('ERROR: Application run failed: ${it.message}')
}
```

### Benefits

- **Clean Shutdown:** Resources properly cleaned up
- **Extensible:** Easy to add more cleanup handlers
- **Observable:** Shutdown progress logged
